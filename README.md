# MicroCourses Assignment

## Overview

Each component is located in its corresponding folder with a installation instructions for each

## Component 1 
### Installation:
```
cd frontend
npm install
```
### How to start:
```
npm start dev
```
### Notes:
Webpage at http://localhost:5173

## Component 2
### Installation:
```
cd backend
npm install
```
### How to start:
```
node --env-file=.env server.js
```
