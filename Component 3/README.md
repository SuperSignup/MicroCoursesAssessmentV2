# MicroCourses Component 3

## Installation:
```
cd backend
npm install
cd ..
cd frontend
npm install
```
Place the server.env file in the backend folder

## Get Started:

### Backend:
```
cd backend
node --env-file=server.env server.js
```
### Frontend:
```
cd frontend
npm run dev
```
Webpage available at http://localhost:5173


## REST API
GET http://localhost:3000/courses/
GET http://localhost:3000/courses/:id/
POST http://localhost:3000/courses/
